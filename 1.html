HTML

Hypertext Markup Language (HTML) is the stadard markup language for creating web pages and web applications. web browsers recieve HTML documents frim a web server of from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document.

HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objets, such as interactive forms, may be embedded into the rendered page. It provides a means to create structed documents by denoting structral semantics for text such as headings, paragraphs, lists, links, quoutes and other items. HTML elemets are delineated by tags, written using angel brackts.
